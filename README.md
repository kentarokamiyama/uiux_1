# かんたんなデザイン入りページ
```
デザイン入りのHTMLページを作って。
ファイルは `frontend/design2.html` として置いて。
内容は以下のとおり。
- サービスの紹介ページです。サービス内容は開発者向けAIツールの導入支援。
- サービス内容は以下の 2本立て。
    - プロフェッショナルによるカスタマイズされた導入支援サービス
    - ベストプラクティスや設定ファイルテンプレートをパッケージにしたかんたん導入キット
- いまどきの開発プロダクトのLPにふさわしいモダンなデザイン
    - three.js を使ったカッコいいエフェクトもほしい
- 開発者や、決裁権を持つ開発マネージャーにとって、導入の決め手となるような情報も盛り込んで。
```
